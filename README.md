[vue-typescript-admin-template](https://github.com/Armour/vue-typescript-admin-template)


基于角色的用户权限管理

用户与角色多对多，角色与菜单权限多对多，当用户所对应的角色拥有某权限时，该用户才可以执行该权限的操作

菜单管理前端页面是一个可拖拽的树形表格，因此在数据交互时，需要传输全部数据
